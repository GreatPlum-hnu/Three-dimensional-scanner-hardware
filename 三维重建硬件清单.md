## 2. 关键参数选择建议

### 雷达

- 优先选择 3D 激光雷达，不建议使用单线 2D 雷达作为主扫描传感器。
- 手持三维重建更看重近距离盲区、视场角、重量和点频。
- 雷达必须能稳定输出点云数据，并且最好支持 PTP、PPS 或 GPS 时间同步。
- 如果用于室内建图，视场角大、近距离盲区小的雷达更合适。

### 相机和镜头

- 相机建议选全局快门，减少手持运动造成的图像变形。
- 如果需要彩色点云，选择彩色工业相机。
- 镜头焦距需要根据扫描距离选择：近距离大场景可选 4 mm/6 mm，中距离可选 8 mm/12 mm。
- 镜头固定后不要频繁调整焦距和光圈，否则需要重新标定内参。

### 时间同步器

- 相机支持外触发时，可以由同步器输出 TTL 脉冲触发曝光。
- 雷达支持 PTP 时，可通过 PTP 主时钟进行网络时间同步。
- 雷达支持 PPS + NMEA 时，可用 PPS 秒脉冲和串口时间报文对齐时间戳。
- 输出电平必须匹配设备要求，常见有 3.3 V TTL、5 V TTL、光耦输入等。

### 结构和外壳

- 雷达和相机的相对位置必须刚性固定。
- 支架材料建议使用铝合金、碳纤维或高强度 3D 打印材料。
- 外壳要兼顾散热、走线、重量和握持舒适性。
- 标定完成后，不要再移动雷达和相机的安装位置。

## 3. 系统连接关系

```text
数据链路：
[ 相机 ] --USB3.0 数据线--------------------> [ 主机 NUC ]
[ 雷达 Livox Mid-360 ] --以太网数据线-------> [ 主机 NUC ]

同步链路：
[ 时间同步器 ] --相机触发/同步信号线--------> [ 相机 ]
[ 时间同步器 ] --雷达同步信号---------------> [ 雷达 Livox Mid-360 ]

供电链路：
[ 酷态科 15 号 Ultra 移动电源 ]
    |--主机供电诱骗线-----------------------> [ 主机 NUC ]
                                                |
                                                |--同步器供电线--------> [ 时间同步器 ]
    |--USB-C / DC 输出----------------------> [ DC 转换模块 / 电源分配 ]
                                                |
                                                |--雷达供电线----------> [ 雷达 Livox Mid-360 ]

结构固定：
[ 雷达 Livox Mid-360 ] + [ 相机 + 镜头 ] --刚性支架固定--> 完成外参标定后不再改变相对位置
```

## 4. 标定件用途

| 标定件 | 用途 | 备注 |
|---|---|---|
| 相机内参标定板 | 标定相机焦距、主点、畸变系数 | 常用棋盘格、圆点板、AprilGrid |
| 雷达相机标定板 | 标定雷达坐标系到相机坐标系的外参 | 需要让雷达点云和图像都能稳定识别标定板 |
| 尺子/卷尺 | 检查标定板尺寸和安装距离 | 用于误差检查 |

## 5. 采购注意事项

- 雷达、相机、镜头、线缆必须确认接口匹配，尤其是雷达一分三线、雷达供电线、主机供电诱骗线和相机信号线。
- 买雷达前要确认供电电压、数据接口、同步方式和 SDK 是否满足项目需求。
- 雷达到货后、安装支架或外壳前，先拍照记录雷达铭牌和序列号；后续配置 MID360 IP、核对设备和排查驱动问题时可能会用到。
- 工业相机要确认是否支持硬触发，否则时间同步只能依赖软件时间戳，误差会更大。
- 酷态科 15 号 Ultra 给 NUC 和传感器供电时，要核算总功率，并预留 30% 以上余量；NUC 通常需要稳定的 19 V/20 V 输入，主机供电诱骗线需要确认触发电压、圆口规格、极性和功率余量。
- Mid-360 供电范围为 9-27 V DC，不能直接随意接 USB 输出；雷达供电线需要按一分三线和同步器方案确认电压、极性、线序和线径。
- 外壳和支架做好后再标定；如果拆装过传感器，需要重新做雷达相机外参标定。
- 淘宝商品链接容易失效，建议用本文档的关键词搜索，并优先选择参数完整、可开发票、支持退换的店铺。
