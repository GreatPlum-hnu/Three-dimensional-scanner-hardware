## 一键启动

完成环境安装、驱动配置、相机内参标定和雷达-相机外参标定后，可以使用启动脚本：

```bash
./scripts/start_fast_livo_mid360.sh
```

当前默认使用 `mapping_avia.launch` 做启动测试。正式 MID360 三维重建时，应将 `FAST_LIVO_LAUNCH` 设置为已经完成标定配置的 MID360 launch 文件。
